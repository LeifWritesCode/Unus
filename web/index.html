Hello, my name is Unus

(That means 'one' in latin!)

I use an Elliptic Curve Augmented Encryption Scheme to enable speedy and secure secret sharing.

Please note that my creator only makes me available for educational purposes, "as is", without warranty of any kind, express or implied.

You can push a new secret by sending a POST request to the /api/v1/secrets endpoint. I support images in JPEG, PNG and GIF format; as well as plain strings.

On success, you'll receive a JSON response similar to the following:

{ "Id": 357420373114880, "Passphrase": "byproduct-Colorado-salespeople-unplugged" }

To retrieve a secret, issue a GET request to /api/v1/secrets/:id, where :id is the ID given to you by your sharer. You should include an Authorization header that includes an HTTP Basic field beginning with Basic, followed by a space, and then the password your sharer gave you in base64 encoding. That would look something like this: Basic bXlzdXBlcnNlY3JldHBhc3N3b3Jk.

Unus is clever. It remembers what was sent to you and returns it to you in the correct form. If you were sent a PNG image, for example, that's exactly what you'll get back.
